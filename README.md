# homebrew-tap

Homebrew tap for kumamaki tools.

## Installation

```bash
brew tap kumamaki/tap
```

## Formulae

| Formula | Description |
|---------|-------------|
| [tunshare](https://github.com/kumamaki/tunshare) | macOS TUI app that shares your VPN connection over LAN via pf NAT |

### tunshare

```bash
brew install tunshare
sudo tunshare
```
