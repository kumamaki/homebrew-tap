# homebrew-tap

Homebrew tap for Mehdi-Hp tools.

## Installation

```bash
brew tap Mehdi-Hp/tap
```

## Formulae

| Formula | Description |
|---------|-------------|
| [tunshare](https://github.com/Mehdi-Hp/tunshare) | macOS TUI app that shares your VPN connection over LAN via pf NAT |

### tunshare

```bash
brew install tunshare
sudo tunshare
```
